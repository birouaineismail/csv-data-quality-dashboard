# CSV Data Quality Dashboard

A web application built with FastAPI, Pandas, HTML, CSS and JavaScript for analyzing CSV datasets.

## Features

- Upload CSV files
- Validate file format
- Count rows and columns
- Display column names
- Detect missing values
- Display data types
- Count duplicate rows
- Error handling for invalid or empty files

## Technologies

- Python
- FastAPI
- Pandas
- HTML
- CSS
- JavaScript
